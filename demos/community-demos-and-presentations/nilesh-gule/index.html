Nilesh Gule - Retrieving RabbitMQ related secret from Azure KeyVault

This demo created by Nilesh Gule is using VMSS Managed Identity and Secret Store CSI provider for Key Vault to retrieve RabbitMQ related secret from Azure Key Vault.

These secrets are synced with Kubernetes Secret object and then injected into deployment as ENV variables.
